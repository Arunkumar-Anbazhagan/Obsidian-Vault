# Topic 5: Functions & Parameters

## 1. Syntax Structure
`ReturnType functionName(Parameters) { Body }`

## 2. Arrow Functions (`=>`)
* **Use Case:** Short, one-line functions.
* **Behavior:** Implicitly returns the result of the expression.
* **Video Ref:** [Arrow Function Syntax [00:09]](https://www.youtube.com/watch?v=YdbWgLQcyrM&t=9s)
	
## 3. Parameter Types
| Type | Syntax | Description |
| :--- | :--- | :--- |
| **Positional** | `(int a, int b)` | Strict order. Must provide all arguments. |
| **Named** | `({int a, int b})` | Flexible order. formatting used heavily in Flutter widgets. |
| **Video Ref:** [Positional vs Named Parameters [00:16]](https://www.youtube.com/watch?v=sFCpoMOdSDE&t=16s)

## 4. Key Keywords
* **void:** The function returns nothing (just performs an action).
* **required:** (Used in Named Params) Forces the user to provide this value.
    * Example: `void login({required String password})`
