1. Numbers (`int`, `double`, `num`)

Dart numbers are subtypes of the `num` class. There are two specific flavors,:

• **int**: Integer values no larger than 64 bits. These represent whole numbers without decimals.

    ◦ _Note:_ On the web, they map to JavaScript numbers (which are floating-point), but on native platforms, they are 64-bit integers.

• **double**: 64-bit (double-precision) floating-point numbers, used when fractional values are needed.

**Key Operations & Examples:** The `num` type includes basic operators like `+`, `-`, `/`, and `*`. Bitwise operators (like `<<` and `|`) are specific to the `int` class,.

```Dart
int x = 1;
double y = 1.1;
num z = 10; // Can hold both int and double

// Automatic conversion
double a = 1; // Equivalent to double a = 1.0 [6]

// Parsing strings to numbers [5]
var one = int.parse('1');
var onePointOne = double.parse('1.1');

// Convert number to string [5]
String piAsString = 3.14159.toStringAsFixed(2); // "3.14"
```
