# Topic 3: Dart Fundamentals

## 1. Typing Systems (Review)
* **Static Typing (Dart):** Strict. You must define what type of data a variable holds. Catches errors *before* the app runs.
* example: var height=1.66;
	* then we cannot change this to height= "height"; //its wrong
* **Dynamic Typing (Python/JS):** Flexible. Variables can hold anything and change types. Catches errors *while* the app runs (crashes).

## 2. Variables
A variable is a container for storing data.
* **Syntax:** `Type variableName = Value;`

## 3. Basic Data Types
| Type | Name | Definition | Example |
| :--- | :--- | :--- | :--- |
| **int** | Integer | Whole numbers (no decimals) | `int age = 25;` |
| **double** | Double | Numbers with decimals | `double price = 19.99;` |
| **String** | String | Text (requires quotes) | `String name = "Alice";` |
| **bool** | Boolean | True or False logic | `bool isActive = true;` |

## 4. Naming Convention
* **Style:** camelCase
* **Rule:** Start lowercase, then capitalize new words. No spaces.
* **Examples:** `myFirstVariable`, `totalScore`, `userEmailAddress`
