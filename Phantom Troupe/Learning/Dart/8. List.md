# Topic 10: Lists (Collections)

## 1. Concept
A way to store multiple values in a single variable.
* **Index:** The position of an item (Starts at 0).
* **Type Safety:** `List<String>` cannot hold numbers.

## 2. Basic Syntax
```dart
List<String> names = ["Alice", "Bob"];
List<int> numbers = [10, 20, 30];
```

3. adding elements to the list
4. ```Dart
   List<double> list1=[100.1,2222.1,2121.3];
   //adding a element to the list
   list1.add=(11.0);
   print(list1);

   print(list1.length());
