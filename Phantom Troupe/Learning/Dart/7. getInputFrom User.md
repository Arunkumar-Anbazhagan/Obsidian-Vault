# Topic 12: Implementing User Input

## 1. The Recipe
To accept user input, you generally follow these three steps:
1.  **Prompt:** Use `print()` to tell the user what to type.
2.  **Read:** Use `stdin.readLineSync()!` to capture their typing.
3.  **Parse (Optional):** If you need math, convert the String to a Number.

## 2. Code Snippet (String vs. Number)
```dart
import 'dart:io';

void main() {
  // 1. String Input
  print("Enter Username:");
  String username = stdin.readLineSync()!;

  // 2. Integer Input
  print("Enter Level:");
  String levelStr = stdin.readLineSync()!;
  int level = int.parse(levelStr);

  // 3. Double Input
  print("Enter Price:");
  String priceStr = stdin.readLineSync()!;
  double price = double.parse(priceStr);
}
```
